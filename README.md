# To-Do List
Página web para criação e controle de tarefas

## Aplicação web para criação e controle de tarefas feita com React-TS

### Tecnologias
- React
- Typescript
- Styled-Components
- React Router DOM

### Como rodar o projeto

1 - Clone o repositório

2 - Instale as dependências:
    
    npm install

3 - Execute o projeto

    npm run dev
